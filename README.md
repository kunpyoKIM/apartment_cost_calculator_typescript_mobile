# 국토부단가 아파트 오피스텔 근린상가

전문가용 모바일 화면 구성으로 재정리한 공사비·분양가 산정 앱임.

## 반영내용
- 상단 파란 영역에 전체 초기화 버튼 및 화면모드 버튼 배치
- 하단 메뉴는 홈/결과 유지
- 기존 입력·계산·결과 항목 유지
- 결과 화면에 토지비, 총사업비, PF지표요약, 공사비구성그래프, 공정률기반자금집행, 예상 최저 분양가 포함
- 노란색 입력란은 모바일 숫자 키보드 우선 표시
